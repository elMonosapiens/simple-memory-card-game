# Simple Memory Card Game

[![Made with Unity](https://img.shields.io/badge/Made%20with-Unity%206.3%20LTS-black?logo=unity&logoColor=white)](https://unity.com/) [![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE) [![Play Online](https://img.shields.io/badge/Play-itch.io-orange)](https://elmonosapiens.itch.io/flip-em-cats)

## About

I started this small project to explore the AI opponent behaviour, its memory feature, and the card-shuffling logic from a minigame I created years ago with a friend. Initially designed as minigame from a larger project of mine, **Mosquintal** (still in development), this version is being expanded into a more robust standalone release: the spin-off **Mosquintal: Memory.**

## Features

- Built with **Unity 6.3 LTS** and **C#**
- Simple game loop
- Loose turn-based mecanic
- Card-shuffling logic
- AI opponent with memory decay

## Dependencies

- [DOTween](https://assetstore.unity.com/packages/tools/animation/dotween-hotween-v2-27676)

## How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/elmonosapiens/simple-memory-card-game.git
   ```
2. Open the project in **Unity Hub** (Unity 6.3 LTS recommended).
3. Install **DOTween**:
   - Open **Unity Package Manager** → **My Assets**
   - Search for **DOTween (HOTween v2)** and install

4. Run the `Game` from the **Scenes** folder.
